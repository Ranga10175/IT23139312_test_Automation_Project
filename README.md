# IT23139312_test_Automation_Project
Automated test suite for Singlish to Sinhala transliteration system using Playwright and Python.

## 📌 Project Overview

This project is developed as part of the ITP assignment to test a Singlish to Sinhala translator web application.

The automation script reads test cases from an Excel file, enters the Singlish input into the translator website, captures the actual Sinhala output, and compares it with the expected output.

## 🎯 Objectives

- Automate Singlish to Sinhala translation testing
- Reduce manual testing effort
- Validate expected and actual outputs
- Identify PASS and FAIL test cases
- Store test results in an Excel file

## 🚀 Features

- Automated UI testing using Playwright
- Excel-based test case reading
- Automatic actual output capturing
- PASS / FAIL result comparison
- Supports positive and negative test cases
- Dynamic column detection in Excel
- Easy result tracking

## 🛠️ Technologies Used

- Python
- Playwright
- OpenPyXL
- Excel
- VS Code
- GitHub
